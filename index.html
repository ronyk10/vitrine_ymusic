Le projet

YMusic est un blind test sous la forme d'un pendu.

Le principe est simple, vous devez deviner le nom de l'artiste ou le titre de la chanson en fonction de la catégorie choisie.

Vous avez 5 essais pour trouver le mot caché.

Si vous ne trouvez pas le mot, vous perdez une vie.

Si les vies tombent à 0, c'est perdu !

Si le mot est trouvé, c'est gagné !

Entre le jazz, le rock, la pop l'électro et bien d'autres, vous trouverez forcément votre style de musique pour vous amuser seul, en famille ou entre amis !
